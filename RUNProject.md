🚀 Quick Start Summary
1. Start Docker Desktop
Ensure Docker is running (whale icon steady in system tray).

2. Launch the project
Open PowerShell in the project root folder and run:

powershell
docker compose up -d
3. Access the app
Open your browser and go to:
http://localhost

4. Log in
Email: admin@pharmacy.com

Password: admin123

Everything will be exactly as you left it – data, settings, and accounts are preserved.

🛑 To stop the project
powershell
docker compose down
